# ProgramaNotas
## Descripción
Este proyecto consiste en un algoritmo que cuenta con la capacidad de recibir una serie de notas académicas y 
determinar si el alumno está suspenso o no. También es capaz de decir si la nota más alta del alumno se repite.
## Guía de ejecución
Basta con instalarse Java en el ordenador y ejecutar el fichero jar.
